# FAQ

## Can I use only Telegram without Web UI

Yes. Core operations are available in the bot.

## Can documentation be opened as a separate page

Yes. There is a standalone docs page that opens from Telegram and in any browser.

## Why does `Reload` fail

Most often service is stopped or config has an error. Start Sing-Box first, then reload.

## Why are logs empty

If container just crashed or is down, logs can be empty in UI. Check status first and restart.

## Should I keep old backups

Yes. Keep several recent backups, not only one archive.

## What should not be translated

Technical product names and terms such as `SSL`, `Nginx`, `Sing-Box` should usually stay as-is for clarity.
